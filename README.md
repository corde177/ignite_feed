#  Ignite Feed 

# Sobre o projecto:

O projeto foi feito para o exercício prático sobre programação, um programa da #Rocketseat, com a trilha de ReactJs. Este projeto tem  a finalidade de gerenciar as publicações em uma rede social.

# Tecnologias Usadas:

- [x]  Vite
- [x]  Css module
- [x]  ReactJs
- [x]  JavaScript
- [x]  Typescript
